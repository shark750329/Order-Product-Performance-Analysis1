# Order-Product-Performance-Analysis1
Created an analytical report using Excel Pivot Tables to explore order patterns and product performance. Aggregated sales data by category, item name, month, and time-of-day to identify growth opportunities and demand patterns.

Work Done:

Cleaned and prepared dataset containing order_id, order_time, item_name, category, price, hours, month, period.

Built pivot tables for:
✔ Sum of Price by Period
✔ Count of Items by Time
✔ Sum of Price by Item Name
✔ Category-wise Revenue
✔ Monthly and Hourly Insights

Designed charts to visually represent sales peaks, category share, and top-selling products.

Tech Stack: Microsoft Excel, Pivot Tables, Charts, Basic Data Analysis
